# RFID-Based Security Control System

## Overview
An RFID authentication system built using the PIC18F4550 
microcontroller for real-time secure access control with 
low false-acceptance rates.

## Tech Stack
- PIC18F4550 Microcontroller
- Embedded C
- MPLAB X IDE
- Custom PCB Design
- Proteus (Simulation)

## Features
- Real-time RFID card authentication
- Secure access control with low false-acceptance rates
- Custom PCB design for compact deployment
- Reliable hardware-software communication via embedded C firmware

## Files Included
- Hex file for flashing directly to PIC18F4550
- MPLAB X project files

## How to Flash
1. Open MPLAB X IDE
2. Load the hex file using IPE (Integrated Programming Environment)
3. Connect PIC18F4550 via PICkit programmer
4. Flash and run
